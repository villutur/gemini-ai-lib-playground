# Gemini AI Agent Instructions

This file contains forward reference instructions for Gemini AI agents working
in the `gemini-ai-lib-playground` repository.

## Primary Instructions

Refer to the main `AGENTS.md` file in this repository for complete guidance on
working with `gemini-ai-lib-playground`.

## Key Points for Gemini Agents

- This is a lightweight playground for trying `@villutur/gemini-ai-lib`
  integrations
- Use `@villutur/gemini-ai-lib` through the package name, not cross-repo source
  imports
- Prefer server-side Gemini access by default
- Use `GEMINI_API_KEY` for server-side Gemini integration

## Repository Context

`gemini-ai-lib-playground` currently uses:

- React + Vite for the client
- Express for the server
- A small full-stack structure meant for experiments and examples
