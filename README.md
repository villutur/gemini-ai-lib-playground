# gemini-ai-lib-playground

Minimal React + Vite + Express starter app for experimenting with
`@villutur/gemini-ai-lib`.

## Scripts

```bash
pnpm install
pnpm dev
```

This starts:

- Vite client on `http://localhost:5173`
- Express server on `http://localhost:3001`

## Included starter pieces

- React client with a small hello-world screen
- Express server with `/api/hello`
- Example use of `@villutur/gemini-ai-lib` model catalogs on the server

## Build

```bash
pnpm build
pnpm start
```
